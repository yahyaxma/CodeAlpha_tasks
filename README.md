# Task 1: Basic Network Sniffer

## Description
This Python script captures network packets and displays the Source IP, Destination IP, and Protocol (TCP, UDP, ICMP). Built as part of my Code Alpha Internship.

## How to Run
Run the script with sudo privileges in Kali Linux:
`sudo python3 my_sniffer.py`
